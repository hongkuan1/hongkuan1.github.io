# 单调性

  • 单调增 f′(x)>0↗\qquad f'(x) > 0 \qquad \nearrowf′(x)>0↗
  • 单调减 f′(x)<0↘\qquad f'(x) < 0 \qquad \searrowf′(x)<0↘
  • 讨论单调性注意点
    • 一阶导数为 0 的点 (驻点)
    • 一阶导数不存在的点

# 凹凸性

  1. 凹曲线: f′(x)\qquad f'(x)f′(x) 递增 f′′(x)>0↑\qquad f''(x) > 0 \qquad \uparrowf′′(x)>0↑
  2. 凸曲线: f′(x)\qquad f'(x)f′(x) 递减 f′′(x)<0↓\qquad f''(x) < 0 \qquad \downarrowf′′(x)<0↓

# 凹凸区间分界点

  1. 二阶导数为零的点
  2. 二阶导数不存在的点

# 间断点

  • 第一类间断点 (可去，跳跃)
  • 第二类间断点 (无穷，振荡)

# 极值

# 定义

  • 邻域内 (局部)，最大 (小) 值 (不唯一)

# 定理

  1. (必要条件) f(x)f(x)f(x) 在 x0x_0x0​ 可导，在 x0x_0x0​ 处取极值，f′(x0)=0f'(x_0) = 0f′(x0​)=0
  2. (充分条件) f(x)f(x)f(x) 在 x0x_0x0​ 连续，左边 f′(x0)>0f'(x_0) > 0f′(x0​)>0 , 右边 f′(x0)<0f'(x_0) < 0 \quadf′(x0​)<0 取极大值
  3. (充分条件) f(x)f(x)f(x) 在 x0x_0x0​ 连续，左边 f′(x0)<0f'(x_0) < 0f′(x0​)<0 , 右边 f′(x0)>0f'(x_0) > 0 \quadf′(x0​)>0 取极大值
  4. (充分条件) x∈U˚(x0)x \in \mathring{U}(x_0)x∈U˚(x0​) 在 f′(x0)f'(x_0)f′(x0​) 符号不变， x0x_0x0​ 不是极值

# 判断方式

  1. 求f′(x)f'(x)f′(x)
  2. 求驻点和不可导的点
  3. 考察左右导数符号是否变化
    • 左边 f′(x0)>0f'(x_0) > 0f′(x0​)>0 , 右边 f′(x0)<0f'(x_0) < 0 \quadf′(x0​)<0 取极大值
    • 左边 f′(x0)<0f'(x_0) < 0f′(x0​)<0 , 右边 f′(x0)>0f'(x_0) > 0 \quadf′(x0​)>0 取极大值
  4. 二阶可导 f′(x0)=0f′′(x0)≠0f'(x_0) = 0 \quad f''(x_0) \neq 0f′(x0​)=0f′′(x0​)=0
    • f′′(x0)<0f''(x_0) < 0 \quadf′′(x0​)<0 在 x0x_0x0​ 处取极大值
    • f′′(x0)>0f''(x_0) > 0 \quadf′′(x0​)>0 在 x0x_0x0​ 处取极小值

# 最值

  • 整个区域内 (全局)，最大 (小) 值 (唯一)
