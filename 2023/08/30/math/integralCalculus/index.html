# 换元积分法

  • 核心思想 (凑), 把 d 前面的某一部分，求原函数拿到 d 的里面去
    • ∫1dx=x+C\int 1 dx = x + C∫1dx=x+C
    • ∫1du=u+C\int 1 du = u + C∫1du=u+C
    • ∫1dF(u)=F(u)+C\int 1 dF(u) = F(u) + C∫1dF(u)=F(u)+C
    • ∫1d(x2−3)=x2−3+C\int 1 d(x^2 - 3) = x^2 - 3 + C∫1d(x2−3)=x2−3+C
  • x=ψ(t)x = \psi(t)x=ψ(t), 先积分出来，然后再换回 xxx
    • 第二类换元思路 (常用是前三项)
    1. a2−x2x=asin⁡(t)→a2−a2sin⁡2(t)=a2cos⁡2(t){\sqrt{a^2 - x^2}} \qquad {x = a\sin(t)} \rightarrow {\sqrt{a^2 - a^2\sin^2(t)}} = {\sqrt{a^2\cos^2(t)}}a2−x2​x=asin(t)→a2−a2sin2(t)​=a2cos2(t)​
    2. x2−a2x=asec⁡(t)→a2sec⁡2(t)−a2=a2tan⁡2(t){\sqrt{x^2 - a^2}} \qquad {x = a\sec(t)} \rightarrow {\sqrt{a^2\sec^2(t) - a^2}} = {\sqrt{a^2\tan^2(t)}}x2−a2​x=asec(t)→a2sec2(t)−a2​=a2tan2(t)​
    3. x2+a2x=atan⁡(t)→a2tan⁡2(t)+a2=a2sec⁡2(t){\sqrt{x^2 + a^2}} \qquad {x = a\tan(t)} \rightarrow {\sqrt{a^2\tan^2(t) + a^2}} = {\sqrt{a^2\sec^2(t)}}x2+a2​x=atan(t)→a2tan2(t)+a2​=a2sec2(t)​
    4. a2−x2x=acos⁡(t)→a2−a2cos⁡2(t)=a2sin⁡2(t){\sqrt{a^2 - x^2}} \qquad {x = a\cos(t)} \rightarrow {\sqrt{a^2 - a^2\cos^2(t)}} = {\sqrt{a^2\sin^2(t)}}a2−x2​x=acos(t)→a2−a2cos2(t)​=a2sin2(t)​
    5. x2−a2x=acsc⁡(t)→a2cos⁡2(t)sin⁡2(t)=a2cot⁡2(t){\sqrt{x^2 - a^2}} \qquad {x = a\csc(t)} \rightarrow {\sqrt{a^2\dfrac{\cos^2(t)}{\sin^2(t)}}} = {\sqrt{a^2\cot^2(t)}}x2−a2​x=acsc(t)→a2sin2(t)cos2(t)​​=a2cot2(t)​
    6. x2+a2x=acot⁡(t)→a2cot⁡2(t)+a2=a2csc⁡2(t){\sqrt{x^2 + a^2}} \qquad {x = a\cot(t)} \rightarrow {\sqrt{a^2\cot^2(t) + a^2}} = {\sqrt{a^2\csc^2(t)}}x2+a2​x=acot(t)→a2cot2(t)+a2​=a2csc2(t)​
