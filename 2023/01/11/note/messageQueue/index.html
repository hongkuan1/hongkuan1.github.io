# 定义

  • 消息队列：消息是进程或是系统之间通讯的一种方式。系统之间通讯还有另一种方式：RPC（目前的gRPC）.

  • 组成部分：

    1. Broker (消息服务器,核心部分)
    2. producer (消息生产者)
    3. consumer (消息消费者)
    4. Topic (主题)
    5. Queue (队列)
    6. Message (消息体)

  • 消息队列

    1. 应用解耦
    2. 流量削峰
    3. 异步通信
    4. 日志收集
    5. 事务最终一致性 ：定义全局事务管理器，和局部资源管理器
