# 关于我

👋 你好！我是 洪宽。

我目前处在职业转型阶段，正在重新梳理自己的方向。过去我接触过技术相关内容，也有写博客、整理知识和做项目的习惯。现在的我更希望找到一个真正适合自己的长期方向，而不是只为了短期稳定随便选择。

# 性格与沟通

我的性格整体比较真诚、直接，可能有一点慢热。刚认识的时候我不一定特别会表现自己，但熟悉之后会比较愿意分享，也会认真对待一段关系。我比较看重沟通、责任感和稳定感，遇到问题时更希望两个人能好好说清楚，而不是互相猜测或冷处理。

# 兴趣与生活

平时我喜欢写一些东西，整理自己的想法，也会关注学习、生活和个人成长。空闲的时候，我可能会散步、看书、找一家咖啡店坐坐，或者研究一些自己感兴趣的话题。我不是特别热闹的人，但我喜欢真诚、有内容的交流。

# 关于感情

在感情里，我希望认识一个真诚、情绪稳定、愿意沟通的人。她不一定要和我兴趣完全一样，但希望对生活有自己的想法，也愿意慢慢了解彼此。我比较期待长期、认真、互相支持的关系，而不是很快开始又很快消耗掉的关系。

# 朋友眼中的我

如果朋友要介绍我，可以简单说：他是一个正在转型阶段、比较真诚和上进的人，有点慢热，但相处起来比较靠谱。他喜欢学习和写东西，也希望认真认识一个可以长期相处的人。
