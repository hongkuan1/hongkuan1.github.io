# 洪宽

正在把过去的经历重新整理成下一段人生的燃料。喜欢学习、记录、复盘，也在认真寻找一个能长期投入的方向。

我现在处在职业转型阶段。比起临时找一个看起来稳定的答案，我更想把技术、表达、项目经验和自我认知重新拼起来，找到真正适合长期发展的路线。

# 当前坐标

维度 现在的状态
方向 从旧路径里抽离出来，重新建立职业定位
习惯 写博客、整理知识、做项目、复盘问题
关注 数学、软件工程、后端开发、系统设计、个人成长
节奏 不追求短期热闹，更重视持续进步和真实反馈

这个博客不是单纯的笔记仓库，更像是一块长期打磨的工作台：把看过的、学过的、踩过的坑，都沉淀成以后能继续调用的经验。

# 我的关键词

真诚 稳定 慢热 上进
不太擅长包装自己，但会认真对待重要的人和事 希望遇到问题时能沟通，而不是冷处理 刚开始可能安静，熟悉后会愿意分享 还在升级系统，也愿意承认自己正在路上

模块 关键词 我正在做的事
技术底盘 Java / 后端 / 软件工程 / 系统设计 把零散经验整理成稳定能力

技术是工具，表达是放大器，复盘是导航。三件事放在一起，才会慢慢形成真正属于自己的能力系统。

# 成长路线

阶段 主题 目标
过去 接触技术，写博客，积累项目和学习记录 把能学的东西先学起来
现在 职业转型，重建方向，梳理个人节奏 不急着乱跑，先把方向看清
接下来 聚焦长期能力，做出可展示作品，形成稳定输出 用作品和记录证明自己的成长

# 我怎么和人相处

我比较看重“把话说清楚”。很多关系真正消耗人的地方，不是问题本身，而是问题出现后不愿意沟通。

我不是特别热闹的人，也不太喜欢夸张的人设。相处里，我更在意真实、可靠、愿意一起面对问题。熟悉之前可能慢一点，但如果确认一段关系值得投入，我会认真、稳定，也愿意把自己的计划和生活慢慢打开。

# 我期待的长期关系

我希望认识一个真诚、情绪稳定、愿意沟通的人。她不需要和我兴趣完全一样，也不需要永远积极外向，但希望对生活有自己的想法，也愿意慢慢了解彼此。

我期待的是长期、认真、互相支持的关系，不是很快开始又很快消耗掉的关系。

# 朋友可以这样介绍我

如果朋友要介绍我，可以说：

他是一个正在转型阶段、比较真诚和上进的人。有点慢热，但相处起来比较靠谱；喜欢学习、写东西，也希望认真认识一个可以长期相处的人。

# 给未来的自己

保持清醒，保持行动。不要急着证明自己，也不要停在原地。把每天能做好的事情做好，方向会在持续输出里慢慢变清楚。
